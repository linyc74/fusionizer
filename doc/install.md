# Install

[Instruction page](https://github.com/suhrig/arriba/wiki/02-Quickstart#installation-using-bioconda)

```bash
conda create -n fusion python=3.12
conda activate fusion

conda install -c conda-forge -c bioconda arriba=2.5.1
```

The command `arriba` and `draw_fusions.R` are both directly executable in the environment.
